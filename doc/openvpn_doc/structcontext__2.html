Detailed Description

Level 2 context containing state that is reset on both SIGHUP and SIGUSR1 restarts.

This structure is initialized at the top of the tunnel_point_to_point(), tunnel_server_udp_single_threaded(), and tunnel_server_tcp() functions. In other words, it is reset for every iteration of the main() function's inner SIGUSR1 loop.
